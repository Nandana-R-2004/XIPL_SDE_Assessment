
# Q5 - Mini Virtual Try-On Web Demo

## How to Run in Google Colab

1. Enable GPU
   Runtime → Change runtime type → GPU

2. Run all notebook cells from top to bottom.

3. Upload:
   - Person Image
   - Garment Image

4. Click **Generate Try-On**.

## Features

- Virtual Try-On using CatVTON
- Q1 Garment Attributes
- Q4 Quality Scores
- Person Detection
- Side Pose Warning
- Seated Pose Warning
- Processing Time
